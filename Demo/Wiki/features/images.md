# Images

This page demonstrates how to embed images in your documents using Markdown and HTML syntax.

## Using Markdown

![Example Image](images/MadProgrammer.png)

## Using HTML

<img src="images/MadProgrammer.png" alt="Mad programmer">

## Markdown with a width constraint

![Example Image](images/MadProgrammer.png){width=10%}

## HTML with a width constraint

<img src="images/MadProgrammer.png" alt="Mad programmer" width="10%">

## Clickable Markdown image

[![Example Image](images/MadProgrammer.png)](https://example.com)

## Clickable HTML image

<a href="https://example.com">
	<img src="images/MadProgrammer.png" alt="Mad programmer">
</a>
